BuildBuddy
BuildBuddy is a simple Python app I put together to help you check if your PC parts actually work together before you start building. It’s just an easy way to pick your components and get some quick compatibility info so you don't run into any headaches halfway through a build.

Cool Stuff It Does
Pick your CPU and GPU: Quickly browse and select your main components.
Check for bottlenecks: See if your CPU and GPU are a good match for each other.
Motherboard compatibility: Make sure your CPU and motherboard actually fit.
RAM checks: Ensure your RAM is compatible with the rest of the system.
JSON-powered: Everything is stored in simple JSON databases for easy management.
Clean UI: Built with a modern interface that isn't a pain to look at.
Search system: Use the searchable selection to find your parts fast.
What’s Supported Right Now
I’ve got the basics covered for:

CPUs: Intel & AMD
GPUs: NVIDIA & AMD
I'm definitely planning to add more hardware to the list in future updates.

Tech Stack
Python
CustomTkinter (for the UI)
JSON (for the data)
A Quick Side Note
I wrote the code myself, but I did use AI to help me out with debugging the tricky parts and putting the databases together.

About the Project
I built BuildBuddy while I was learning Python and trying to level up my coding. I just wanted to create a helpful assistant that makes picking parts for a PC build a lot easier and more straightforward.
