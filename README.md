#BuildBuddy
BuildBuddy is a simple Python app I put together to help you check if your PC parts actually work together before you start building.

It’s just an easy way to pick your components and get some quick compatibility info so you don't run into any headaches halfway through a build.

#Cool Stuff It Does
Pick your CPU and GPU
Check for bottlenecks between the CPU and GPU
Make sure your CPU and motherboard actually fit
RAM compatibility checks
Everything is powered by JSON databases
Clean, modern UI that isn't a pain to look at
Easy search to find your parts fast
#What’s Supported Right Now
I’ve got the basics covered for:

Intel and AMD CPUs
NVIDIA and AMD GPUs
I'm definitely planning to add more hardware to the list in future updates.

#Techs That I Used
Python
CustomTkinter
JSON
#A Quick Side Note
I wrote the code myself, but I did use AI to help me out with debugging the tricky parts and putting the databases together.

I built BuildBuddy while I was learning Python and trying to level up my coding. I just wanted to create a helpful assistant that makes picking parts for a PC build a lot easier and more straightforward.
